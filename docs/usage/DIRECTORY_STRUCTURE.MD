## Directory Structure

Here is the suggested directory structure for a Backk microservice:

```
- src
  |- services
  |  |- common
  |  |- service1
  |  |  |- errors
  |  |  |- types
  |  |  |  |- args
  |  |  |  |- entities 
  |  |  |  |- enum
  |  |  |  |- responses
  |  |- service2
  |  |- .
  |  |- .
  |  |- serviceN
  |- microservice.ts
  |- main.ts
```
